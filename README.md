# Papineau Car Detailing Website

A static, single-page website template for a car detailing service based in Winnipeg, Manitoba.

## Features
- **Theme:** Dark theme with Tyrian Purple accents and subtle glassmorphism effects.
- **Components:** Built with plain HTML, CSS, and JavaScript. Uses native `<dialog>` elements for popups.
- **Core Functionality:** Includes a dynamic pricing calculator based on vehicle size, base service, and add-ons. 
- **Responsive:** The layout adapts to mobile devices, and side banners hide on smaller screens.

## Structure
- `index.html`: Main structure and content.
- `style.css`: Styling, layout, and theme variables.
- `script.js`: Dynamic pricing calculation and popup toggling logic.
- `images/`: Directory containing static image assets used in banners.

## Usage
No build step is required. Serve the files directly with any static web server (e.g., `npx serve`, Live Server, or Python's `http.server`) to view the application locally.
